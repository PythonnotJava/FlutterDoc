key : 用于标识组件的唯一值，主要用于在组件树中查找和控制特定组件实例。
value : 进度指示的值，取值范围是 [0.0, 1.0]。如果设置了该值，进度条会显示具体的进度（定量进度条）；如果不设置该值，则为不确定进度条，表示加载状态。
backgroundColor : 指定进度条背景圆环的颜色，即未完成部分的颜色。默认为透明。
valueColor : 前景颜色的动画对象，可以用 AlwaysStoppedAnimation<Color> 来指定静态颜色，也可以通过动画效果实现渐变效果。
strokeWidth : 指定圆环的宽度，默认为 4.0。值越大，圆环越粗。
semanticsLabel : 无障碍支持的标签，用于描述当前的进度条，便于屏幕阅读器等辅助技术理解组件的功能。
semanticsValue : 无障碍支持的数值描述，主要用于说明当前进度的数值，帮助辅助技术提供更好的用户体验。
strokeCap : 指定圆环端部的样式。可以选择 butt、round 或 square，默认值为 StrokeCap.butt。这影响圆环的外观，特别是圆环的两端样式。
